# python-2-dx


Narzędzie do zarządzania ofertami Allegro przez REST API. 

Funkcje:
- tworzenie ofert,
- aktualizacja cen,
- aktualizacja stanów magazynowych,
- wznawianie ofert,
- kończenie ofert,
- pobieranie statusów ofert.

Aplikacja używana lokalnie do integracji z Allegro REST API.

Technologie:
- Python
- Allegro REST API
